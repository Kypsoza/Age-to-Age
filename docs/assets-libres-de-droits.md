## Comment je les intégrerais

1. **Tu télécharges** les packs qui t'intéressent (zip depuis Kenney ou itch.io).
2. **Tu m'envoies** les fichiers PNG/spritesheets ici (upload direct dans le chat).
3. Je les découpe, je les mappe aux bâtiments/tuiles existants dans le code, et je remplace les emojis par les vrais sprites — en gardant la palette de couleurs actuelle pour la cohérence visuelle entre les ères.

**Note sur les licences** : tout ce qui est CC0 ne nécessite aucune mention. Pour les packs "gratuit / pay what you want" et CC BY (game-icons.net, Neo Zero), il faudra une petite page de crédits dans le jeu (souvent juste un fichier `CREDITS.txt` ou un écran "à propos") — je peux te la générer automatiquement une fois qu'on sait lesquels tu as retenus.
